Sistema de presupuestos de obras civiles

Bienvenido al sistema de presupuestos de obras civiles.

A continuación encontrará información relativa al sistema y al uso del mismo.

Descripcion del sistema

El sistema proporciona un entorno web de elaboración de presupuestos sencillo y completo, de interfaz amigable y fácil de utilizar.
Incluye módulos especiales para el mantenimiento de la información de proveedores, herramientas, usuarios, empleados, materiales, mano de obra, y unidades de medida.

Funciones principales

A través del sistema, y dependiendo del tipo de usuario que se trate, usted podrá realizar las siguientes funciones:

  • Usuario: generar presupuestos, gestionar herramientas y proveedores

  • Administrador: gestionar Rubros/subrubros,  usuarios, empleados, materiales, mano de obra, unidades de medida

Inicio y cierre de sesión

Al ingresar al sistema, se le presentará la siguiente pantalla de autenticación.

 

En la misma usted deberá ingresar su nombre de usuario y contraseña proporcionados por el administrador.
Dependiendo del tipo de usuario que se trate, el menú contará con diferentes opciones.  Mas adelante se detallan las opciones disponibles para cada caso.
